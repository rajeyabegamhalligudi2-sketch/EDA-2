# EDA Project
# 🏙️ Dubai Real Estate EDA Project

## 📌 Overview

This project focuses on **Exploratory Data Analysis (EDA)** of the Dubai real estate market. The goal is to uncover patterns, trends, and insights related to property prices, locations, property types, and other key factors influencing the market.

---

## 🎯 Objectives

* Understand the distribution of property prices across Dubai
* Identify key factors affecting real estate prices
* Analyze trends by location, property type, and amenities
* Detect outliers and anomalies in the dataset
* Provide actionable insights for investors and stakeholders

---

## 📂 Dataset

The dataset includes information such as:

* Property Type (Apartment, Villa, Townhouse, etc.)
* Location (Area/Community)
* Price
* Size (sq ft)
* Number of Bedrooms & Bathrooms
* Amenities
* Listing Type (Rent/Sale)


---

## 🛠️ Tools & Technologies

* **Python**
* **Pandas** – Data manipulation
* **NumPy** – Numerical computations
* **Matplotlib & Seaborn** – Data visualization
* **Jupyter Notebook** – Analysis environment

---

## 🔍 Key Analysis Performed

* Data cleaning and preprocessing
* Handling missing values
* Univariate and bivariate analysis
* Price distribution analysis
* Location-wise price comparison
* Correlation analysis
* Outlier detection

---

## 📊 Sample Insights

* Prime locations like Downtown Dubai and Dubai Marina tend to have higher property prices
* Villas generally cost more than apartments per unit
* Strong correlation between property size and price
* Certain areas show high price variance indicating diverse property types

---

## 📈 Visualizations

* Price distribution histograms
* Boxplots for outlier detection
* Heatmaps for correlation analysis
* Bar charts for location-based comparisons
